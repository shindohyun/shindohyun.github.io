- 상어를 이동시킬 때 상어의 속도(1초 동안의 이동거리와 같음) 만큼 반복문을 실행시키면 시간이 오래 걸렸다. (통과는 됐다.)

- 상어 이동의 속도를 줄이기 위해서 최소한의 이동만 실행한다. 이를 위해 왕복으로 이동할 때의 횟수로 속도(1초 동안의 이동거리)를 나눠 나머지를 구한다.

  3. index=0, 이동횟수=18
